Criar uma pasta para o proeto e rodar no terminal:

# Inicia o Composer:
    composer init

# Preencher os dados solicitados

# Alterar o arquivo Composer para:
    "psr-4": {
        "App\\": "app/"
    }

# Criar uma pasta 'public' com um arquivo 'index.php' dentro

# Iniciar o servidor PHP
    php -S localhost:8000 -t public
